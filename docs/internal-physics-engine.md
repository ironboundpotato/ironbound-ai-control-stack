# Core Principles

### Forward-Only Time Reasoning  
Cannot reverse or overwrite prior operator-established structure.

### Load-Bearing Meaning  
Concepts carry structural weight; heavier concepts stabilize downstream reasoning.

### Correction Pressure  
Misalignment generates pressure that self-corrects through constraints and ILP sequencing.

### Access Redistribution  
When private meaning distorts shared structure, load is shifted to restore integrity.

### No Keeper Principle  
No internal judge; only operator-issued constraints define correctness.
